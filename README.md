# Ejecutar en desarrollo

1. Clonar el repositorio

2. Ejecutar
```
npm i 
```
3. Tener istalado Nest CLI
```
npm i -g @nest/cli
```
4. levantar la base de datos
```
docker compose up -d
```
5. Recargar base de datos con la semilla
```
http://localhost:3000/api/v2/seed
```

## Stack Usado
* MongoDB
* Nest
